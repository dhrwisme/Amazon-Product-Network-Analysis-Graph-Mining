# Amazon Product Co-purchase Network: Topology and Community Structure Analysis

## 📌 Project Overview
This project involves a comprehensive Graph Theory analysis of the Amazon product co-purchase network. By modeling recommendation flows as a directed graph, the study identifies structural patterns, influence nodes, and semantic community clusters. This analysis serves as a foundation for understanding complex spatial dependencies in recommendation systems.

## 🚀 Key Research Findings
- [cite_start]**Graph Construction:** Engineered a parsing pipeline for large-scale metadata to construct a directed graph where nodes represent products and edges represent recommendation flows[cite: 56].
- [cite_start]**Network Topology:** Analyzed global metrics including a **Global Clustering Coefficient of 0.23** and Average Path Length to characterize "Small World" properties.
- [cite_start]**Community Detection:** Implemented the **Greedy Modularity Optimization (Louvain algorithm)** to identify distinct structural communities within the "Books" category.
- [cite_start]**Semantic Validation:** Validated that structural clusters strongly correlate with semantic subcategories (e.g., "Fiction", "Business"), demonstrating the effectiveness of topology-based recommendation logic[cite: 58].
